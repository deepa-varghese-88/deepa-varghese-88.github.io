Boba Tea Store Locator

A serverless web app made possible by GCP's AppEngine that lets customers find the best boba tea stores around them, nutrition info for different options and if it's a good weather to go outside. The six ML APIs that were integrated: 1) GCP's Vision API, 2) GCP's Places API, 3) GCP's NLP API, 4) GCP's Translate API, 5) AccuWeather's Weather API, and 6) Nutritionix API
